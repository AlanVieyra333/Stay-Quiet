\subsection{IU1.6 Agregar Protegido}

\subsubsection{Objetivo}
	El usuario agregar la información necesaria para el registro del Protegido.

\subsubsection{Diseño}
	Se despliega al dar clic en el \IUbutton{Agregar contacto} de la pantalla \IUref{IU1.3}{Pantalla Inicial}.

\IUfig[.3]{IU1-6}{IU1.6}{Agregar Protegido}

\subsubsection{Entradas}
Los datos del usuario requeridos:
\begin{itemize}
	\item Nombre completo.
	\item Correo Electrónico.
	\item Numero de Celular.
	\item Contraseña.
\end{itemize}

	\item \MSGref{MSJ1.2}{Formato de nombre incorrecto}

	\item \MSGref{MSJ1.4}{Formato de contraseña incorrecto}
	\item \MSGref{MSJ1.5}{Contraseña no coincide}
	\item \MSGref{MSJ1.6}{Error en conexión}
	\item \MSGref{MSJ1.8}{Datos utilizados}
	\item \MSGref{MSJ1.10}{Formato de correo incorrecto}
\end{Citemize}
